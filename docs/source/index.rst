ELLIPTIc's documentation
========================

**ELLIPTIc**, The ExtensibLe LIbrary for Physical simulaTIons, is a library / framework for creating reusable and extensible
`Domain Specific Languages (DSL) <https://martinfowler.com/bliki/DomainSpecificLanguage.html>`_ for scientific purposes.
